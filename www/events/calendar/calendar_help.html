Adding an event

  • On the calendar page, click "Add new item"
  • Optional: Choose a suitable template, and click "Load"
  • Set the entry type to "calendar".
  • Enter other information - location, time, etc. The "status" field is not normally displayed, and can be used for notes on scheduling, etc.
  • Click "Update Entry".

Adding a new template

  • On the calendar page, click "Add new item"
  • Under Choose Template click Create New Template.
  • Ensure the entry type is set to "template".
  • Enter a category code (usually the initials of the template title). The category code will be stored with all events based on this template, and allow them to be selected by the calendar menus.
  • Enter other information - location, time, etc.
  • Set the date to be on the correct day of week for the repeating event. When the template is used for a new event, the date will be set to the same day in the following week.
  • Click "Update Entry" to store the template.
Example of template dates

The date in a template is set to 16th Sept 2003, which was a Tuesday. On Friday 10th October 2003, you add a new entry using this template. The template will set the date to 14th October 2003 (the following Tuesday).

Editing or deleting a template

  • On the calendar page, click "Add new item"
  • Choose the template and click "Edit". This brings the given template into the form for editing.
  • Either make the changes to the template, and click "Update Entry", or Click "Delete Entry" to remove the template.
