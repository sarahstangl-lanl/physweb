Introduction

The class pages can automaticaly turn simple wiki-style markup into a nice looking web page for you. To use the wiki style you do not need to learn html, only this simplified markup syntax:

Headings

Headings are created by surrounding a line of text with equals (=) and a space.

Example:
= This is a level 1 header =
== This is a level 2 ==
=== This is a level 3 ===

This is a level 1 header

This is a level 2

This is a level 3

Bold, Italic, Underlined

To make some text bold, simply surround it in astrisks (*) like this: *bold*

To make some text italicized, simply surround it in 2 single quotes (') like this: ''italic''

Example:
bold
italic

Lists

Lists can be done like this:

                   * unordered one
                   * unordered two

                   1. ordered one
                   2. ordered two
                               a. nested one
                               b. nested two

Linking

For links, enclose the URL within square brackets:

               [http://www.google.com]
http://www.google.com

Or, with different text:

               [http://www.google.com|Example Link]
Example Link
